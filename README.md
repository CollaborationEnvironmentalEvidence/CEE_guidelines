# CEE_Guidelines


*Guidelines and Standards for Evidence Synthesis in Environmental Management
VERSION 5.1*

This repo contains a sample chapter (Chapter 8) and demonstrates the potential workflow for updating the guidelines.

There are issue templates to help contributors make a contribution and there is a [contribution guide](https://github.com/DrMattG/CEE_Guidelines/blob/main/contribution.md).

On acceptance of this approach for updating the CEE Guidelines I will move this repo under the control of CEE (in an organisation on GitHub).
